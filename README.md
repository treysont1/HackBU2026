# StudBud

## Project Description

"Work in progress" is a web application designed to help students quickly find and join study groups happening around them in real time. Many students struggle to find people studying the same subject at the same time, especially outside of organized classes or scheduled study sessions.
The platform allows users to start a study session by selecting a subject and optionally sharing their location. Active study sessions appear on the dashboard, allowing students to discover and join groups instantly. Users can see what subjects others are studying, how many people are in each session, and join the ones that match their needs.
